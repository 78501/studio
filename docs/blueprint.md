# **App Name**: MeshConnect

## Core Features:

- User Login: Implement three distinct login types: user, medic, and hospital.
- Role-Based Messaging: Enable users to send help messages, and medics/hospitals to view these messages.
- Bluetooth Mesh Networking: Establish a peer-to-peer Bluetooth mesh network for offline communication.
- Offline Messaging: Ensure all messaging functionality operates completely offline within the mesh network.
- Message Persistence: Persist messages locally on each device within the mesh network until delivered.
- Location Sharing (Optional): Allow users to optionally share their location via Bluetooth for faster assistance.

## Style Guidelines:

- Primary color: Calming blue (#64B5F6), evoking trust and stability.
- Background color: Light gray (#EEEEEE), providing a clean and neutral base.
- Accent color: Energetic orange (#FFAB40), highlighting calls to action and urgent messages.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a modern and warm feel, suitable for readability in various conditions.
- Use clear, recognizable icons for message types, user roles, and connectivity status.
- Design a simple, intuitive interface for sending and receiving messages, prioritizing ease of use.
- Incorporate subtle animations to indicate message sending/receiving and network connectivity.
